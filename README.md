# 📊 Alura Store - Análise de Dados

Projeto de análise de dados desenvolvido no desafio de **Data Science da Alura**, utilizando Python e Pandas para explorar informações de vendas de quatro lojas de uma empresa fictícia.

---

# 🎯 Objetivo

Realizar uma análise exploratória dos dados para entender:

- faturamento das lojas
- desempenho de categorias
- avaliação dos clientes
- produtos mais vendidos
- custo médio de frete

A partir dessas análises, identificar **insights que possam apoiar decisões estratégicas de negócio**.

---

# 🛠 Tecnologias utilizadas

- Python
- Pandas
- Matplotlib
- Folium
- Jupyter Notebook

# 📊 Análises Realizadas

## 1️⃣ Faturamento por Loja

A análise mostrou que:

- **Loja 1 possui o maior faturamento**
- **Loja 4 possui o menor faturamento**

---

## ⭐ Avaliação Média das Lojas

| Loja | Avaliação |
|----|----|
| Loja 1 | 3.97 |
| Loja 2 | 4.03 |
| Loja 3 | 4.04 |
| Loja 4 | 3.99 |

A **Loja 3 apresenta a melhor satisfação dos clientes**.

---

## 📦 Produtos Mais Vendidos

Top produtos:

- Cômoda
- Carrinho controle remoto
- Micro-ondas
- Bateria
- Cama king

---

## 🚚 Frete Médio

| Loja | Frete Médio |
|----|----|
| Loja 1 | R$ 34.69 |
| Loja 2 | R$ 33.62 |
| Loja 3 | R$ 33.07 |
| Loja 4 | R$ 31.28 |

A Loja 4 possui o **frete médio mais barato**.

# 📈 Principais Insights

- Loja 1 lidera em faturamento
- Loja 3 possui melhor avaliação
- Loja 4 possui menor faturamento
- Produtos domésticos têm maior volume de vendas
